# Startup Jobs Radar

Welcome to the Startup Jobs Radar project!

## Project Overview
This project aims to aggregate job opportunities in the startup ecosystem, providing a platform for job seekers to easily find and apply for roles at innovative companies.

## Features
- Job Listings: Browse available job positions from various startups.
- Easy Application Process: Apply for jobs directly through the platform.
- Notifications: Get notified when new jobs matching your profile are posted.

## Getting Started
To get started with the project:
1. Clone the repository: `git clone https://github.com/schenkel94/startup_jobs_radar.git`
2. Install dependencies: `npm install`
3. Run the project: `npm start`
